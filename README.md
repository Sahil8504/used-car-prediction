Used Car Price Prediction System

An end-to-end Machine Learning web application designed to predict the selling price of used cars based on multiple features such as brand, model, mileage, age, fuel type, engine size, and transmission.

This repository contains:

Machine Learning Pipeline: Data preprocessing and model training using scikit-learn (Linear Regression model).
REST API Backend: Built with Flask to serve model predictions.
Frontend Client: Built with React to provide an interactive, user-friendly form interface.

Frontend
React.js - UI Component Library
React-Select - Modern, flexible select dropdown menus
CSS3 - Custom UI/UX styling
Backend & Machine Learning
Python 3.x
Flask - Lightweight backend server
Flask-CORS - Handling Cross-Origin Resource Sharing
Scikit-Learn - Machine learning library (StandardScaler, OneHotEncoder, ColumnTransformer, LinearRegression)
Pandas & NumPy - Data analysis and preprocessing
Pickle - Serialization of model and transformer pipelines
